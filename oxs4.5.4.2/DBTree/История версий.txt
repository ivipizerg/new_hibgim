Библиотека DBTree
	Предназначена для работы с деревом NESTED SETS. Дерево с двумя индексами.
php методы
	Init():
		table - имя таблицы в базе данных
		db - обьект базы данных
	ChekForError() - Провести провеку целостности таблицы. Проводит 5 тестов, результат кидает в канал библиотеки.
	
	
	GetTree($Param=NULL) - Возвращает сгенерированный sql запрос на получение дерева
        $Param["foo"]  - фунция в которую будет передан обьект базы данных с начальынм запросом, его можно дополнить и записать в обьект бахы данных, так же передаеться $Param["Params"]
	GetTreeEx($Param=NULL) - Вызыввает GetTree и сразу выполняет сгенерированный запрос
	
	GetChilds($id,$Param=NULL) - возвращает sql запрос на получение потомков узла
        $id = id узла для котоого ищем детей
        $Param["foo"]  - фунция в которую будет передан обьект базы данных с начальынм запросом, его можно дополнить и записать в обьект бахы данных, так же передаеться $Param["Params"]
	GetChildsEx($id,$Param=NULL) - Вызыввает GetChilds и сразу выполняет сгенерированный запрос
	
	GetParents($id,$Params=NULL) - возвращает SQL запрос на получение родителей узла
	GetParentsEx($id,$Params=NULL) - возвращает родителей узла
	
	GetBranchEx($id,$Params=NULL) - Возвращает всю ветку узла
	GetBranch($id,$Params=NULL) - Возвращает сегенированный запрос на поулчение всей ветки

	CreateRoot() - Подготовить пустую таблицу к работе, создать корневую запись
	
	Insert($Parent=0,$Data=NULL) - Вставить новый узел, $Data - ассоциатвиный массив каждый ключь 
это поле в базе данных. в ключах работают флаги проверки данных int float sql number
	Remove($id) - удаляет узел из деерва. Все потомки так же будут удалены!
	ChangeParent($id,$id_move) - Изменить родителя узла с id = $id на родителя с id = $id_move
	Move($id,$id_move,$change_parrent=false) - перемещает узел и всю его ветку с id  = $id в ветку в которой находиться узел с id = $id_move. Вставка произведеться ниже по списку. $change_parrent - если true будет произведена принудительная замена родителя. $change_parrent = true то же самое что ChangeParent
