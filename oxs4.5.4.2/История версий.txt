Фреймворк OXANA
    
    Для удобсат был создан класс интерфейс статических функций, так же методы можно вызывать обращаясь к функции Oxs(), например Oxs()->SetRoot("someroot/");

php методы:

    Oxs::SetRoot($Path = NULL) - Установить корень сайта, относительно которого будут подлючаться все файлы и билиботеки. Необходимо вывзать первым перед инициализацией, если корень сайта не являеться путем к текущему скрипту, например скрипт вызываеться через аякс. Если не передать параметра $Path, то в качестве рута будет принят путь к текущему скрипту. Если передать /(слеш) то за корень будет считаться домен сайта. Для корректной работы JS скриптов необходимо подключить базу Oxs::G("dom")->ShowBase(). По умолчанию базой делаеться текщий каталог.
    
    Oxs::GetRoot() - получить текущий рут 
    
    Oxs::GetBack() - получить обтарный путь до рута. Например если рут somesite.ru/, а скрипт исполняеться по пути somesite.ru/admin/ajax/somescript.php то в нем бэк путь будет ../../
    
    function __construct() - вызывает Oxs::SetRoot();
    
    Oxs::Init(...$Path) - Фукнция инициализации мест хранения билиотек - наборы библиотек. Каждый параметр это путь к набору , относительно рута. В каждом наборе  должен быть создан файл <название набора>.php в котором должен быть создан класс унаследованый от класса lib:
        
    class SomeLib extends Lib{
		
		function __construct($Path){
			parent::__construct($Path);
		}	
	}
	
	А так же два класса для мультибилиотек и одинчоных билиотек
	
	class SomeMultiLib extends SomeLib{

		protected $Type="multilib";
		
		function __construct($Path){
			parent::__construct($Path);			
		}

		function Init($Param=NULL){
			return 0;
		}
	}

	class SomeSingleLib extends SomeLib{

		protected $Type="singlelib";
		
		function __construct($Path){
			parent::__construct($Path);			
		}
	}
	
	Все библиотеки в наборе буду наследоваться от соответсвуещего класса(SomeMultiLib для мультибилиотек и SomeSingleLib для одиночных билиотек) и получать соотвествующие свойства.
	
	Oxs::Error($Text) - Вызывает прерывание с сообщением $Text)
	
	Oxs::LoadLib($Name,$Param=NULL) - Загружает новый экземпляр билиотеки в память и возвращает её обьект. Если  массив $Param не пуст будет вызвана функция Init($Param) билиботеки в которую будет передан этот массив. Если попытаться создать новый обьект одинчной билиотеки программа перкратиться с ошибкой.
	
	Oxs::LoadLib($Name) - вохвращает обект одиночной билиотеки. . Если попытаться получить обьект на мультибилиотеку программа перкратиться с ошибкой.
	
	Oxs::ShowErrors($Mode="default") - ВЫвести отладочное окно. $Mode - каналы котоыре необходимо вывести. По умолчнаию все каналлы.
