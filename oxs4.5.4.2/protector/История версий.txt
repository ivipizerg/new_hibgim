Плагин protector
	
	Плагин для генерации и првоерки защитных токенов при работе с ajax




php методы:
	
	function SetToken($name=NULL) - Создать токен с именем $name
, если не указать имя то оно будет сгенерировано случайным образом и возвращено
	function GetToken($name) - получить токен по имени
	
	function CheckToken($name=NULL,$token=NULL) - проверить токен $name на сотвтетсвие, передаеться название токена, занчение токена. Если не передать $name, оно будет взято из $_POST["TN"], если не передать $token, оно будет взято из $_POST["T"] именно эти занчения заполняет библиотекка js.protect_ajax
