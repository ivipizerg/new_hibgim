Бибилотека calendar
	
	Работа с датами и временем, имеется поддерждка часовых поясов

php методы:

	function Set($Data=NULL) - создать обьект даты.
		IN:
			$Data - строка даты в формате d.m.YYYY H:MM:SS. например 3.12.2017 12:00:00. Передавать можно только дату или тольк овремя. Переданная информация передаеться в часовом поясе +0. Что означает что если устанволен глобальный часовой пояс +3 а вы передали дату указанную выше, она сразу преобразуеться в 15 часов. Поддерживаеться формат строки даты MYSQL YYYY-MM-DD HH-MM-SS. 
	
	static function SetGlobalUtc($Value) - устанвоить глобальный часовой пояс, если дана при инициализации обьекта не указана он будет применен к текщему поясу сервера, например если на сервере 19:00 и пояс +3 то устанвоив глобальынй пояс в 0 мы будем поулчать 16:00 при создании обьектов.
		IN:
			$Value - Пояс, число.

	static function GetGlobalUtc() - Получить длобальный часовйо пояс
		OUT:
			int - пояс
	
	static function Server() - драйвер для работы с временем непосредственно сервера, возвращает обьект calendar с настроенным на серверм временеми часовым поясом.
		OUT:
			calendar - обьект билиотеки с настроенным под сервер данными
	
	function SetUTC($Value) - Устанвоить часовйо пояс, но не пересчитывать дату
		IN:
			$Value - Пояс, число.

	function UseUTC($Value) - Устанвоить часовой пояс и пересчитать дату относительно текущего часовог опояса.
		IN:
			$Value - Пояс, число.

	function GetData()/GetDataMysql() - Получить текущую дату в формате DD.MM.YYYY/YYYY-MM-DD
	
	function GetTime() - Получить текущую дату в формате HH:MM:SS.

	function GetDataTime()/GetDataTimeMysql() - Получить и дату и время в формате DD.MM.YYYY HH:MM:SS/YYYY-MM-DD HH:MM:SS

	function GetUTC() - получить текущий часовой пояс

	function GetHours() - получить текущий час

	function GetMinuts() - текущая минута

	function GetSeconds() - текущая секунда
	
	function GetCountDays() - Получить количестов дней в текущем месяце
	
	function GetWeekDay($Mode=1) - Получить текущий день недели
		IN:
			$Mode - 1 - число,  2 - Полное название, 3 - краткое
	
	function GetMount() - месяц

	function GetDay() - день месяца

	function GetYear() - год
	
	function GetMountName( $mode = false) - название текущего месяца
		IN:
			$mode - true - родительный падеэ false - именительный
	
	function Hour($Hour=0) - изменить час на переданное число

	function NextHour($Hour=1) - прибавть часы к текущему времени
	
	function PrevHour($Hour=-1) - отнять часы от текущего времени

	function NextYear($Year=1) - прибавить год

	function PrevYear($Year=1) - отнять год

	function NextMount($Mount=1) - прибавить месяц
	
	function PrevMount($Mount=1) - отнять месяц

	function NextDay($Day = 1) - прибавить день
	
	function PrevDay($Day = 1) - отнять день

	function CompareData(calendar $Cl) - сравнить даты, равны они или нет

	static function GetMountList() - Получить массив месяцев array ("name" , "id" )
