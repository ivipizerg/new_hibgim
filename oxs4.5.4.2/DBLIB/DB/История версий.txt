Бибилиотека DB
	Работа с базой данных mysql.

php методы:
	
	function SetQ(...$Query) - Установить запрос к базе данных. Комбинация #__ будет автоматический заменена на префикс. Ранее заданный запрос буде тпрезаписан. Функция осуществляет проверку параметров перед выполнением запроса доступны 4 флага проверки int float number и sql. int проверяет параметр на целоче число, float соответсвенно на число с плавющей точкой, number это и int и float - любое число, флаг же sql заставляет функцию выполнить проверку функцией mysqli_real_escape_string(), которая выполнит безопасное экранирвоание параметра. для того что бы задать флаг нужно использовать коминацию oxs: и типа флага: например oxs:sql. Сами же значения будут браться из следующих параметров функции например SetQ("SELECT * FROM `name` WHERE `id` = 'oxs:int'",2).
	function AddQ(...$Query)  - аналогична функции SetQ за исключением того что предыдущий рапрос не будет очищен и будет дополнен.
	Function Exec(...$Query) - выполнить заданные ранее запрос или переданный.
	function GetLastID() - получить id последнего добавленного элемента	
	function GetQ() - получить составной запрос
	function IfEmpty($Table) - проверяет на пустоту таблицу $Table
	function Insert($Table,$Data) - Выполнить запрос вставки в базу даных. Параметр $Table это таблица в которую вставляем, массив дата это массив название поля - занчение, типа поля указываеться перед названием через двоеточие например int:name что означает данные будут вставлены в поле name и пердварительно првоерены являються ли они целым числом.
