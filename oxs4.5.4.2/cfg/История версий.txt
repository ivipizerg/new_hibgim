Билиотека 
	Работа с файлами конфига, запись и счиытваие.



методы:
	
	function SetCfgFile($File) - устанвоить файл конфигурации
	
	function Get($Name,$PathCfgFile=NULL) - получить параметн $Name из файла конфигурации $PathCfgFile, если он не указан то будет использваться файл конфигурации указанный через конструктор или через SetCfgFile.
	
	function Set($name,$val,$PathCfgFile=NULL) - записать занчение $val в параметр $name, в файле конфигурации $PathCfgFile, если он не указан то будет использваться файл конфигурации указанный через конструктор или через SetCfgFile. Значения перезаписываются.
