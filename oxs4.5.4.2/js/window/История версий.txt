Библиотека js.window
	SingleLib Библиотека для создания псевдоокон посредстов js и css

php методы:

	GetObject($Name="js_window",$Param=null) - Подключить все необходимые js файлы и создать обьект окна с именем $Name, а так же передать в этот обьект массив параметров $Param
		$Name - Имя вновб созданного js обьекта окна
		$Param - Массив параметров
			0 - "true"/"false" - включить/выключить отладку	
	
	Inlcude($Param=null) - толькподключить все необходимые js файлы и не создавать обьект окна
		$Param - Массив параметров
			0 - "true"/"false" - включить/выключить отладку	только у панели окон(window_bar)

js класс oxs_js_window:
    
    При создании обьекта окна, с ним же создаеться обьект его событий, а так же при создании первого окна создаеться панель для сворачианя по умолчанию. Панель можно переназначить создав новый обьект панели:
        var new_bar = new oxs_js_window_window_bar("");
    и затем прикрепить окно кновй панели:
        this_window.setNewWindowBar(new_bar);

	set(Data) - устанвоить в окно html информацию
	get() - получить html содержимое окна	
	setWidth(Value) - Задать ширину окна
        	Value - шириниа окна. Аналогичен css параметру width
    	setHeight(Value) - Задать высоту окна
        	Value - высота окна. Аналогичен css параметру height    
    	addClass(calss_name) - добавить класс к оснвоному блок окна  
    	removeClass(calss_name) - удалить класс к оснвоному блок окна    
	setName(Name) - задать название окну, используеться когда оконо свернуто
	stick(value1,value2) - отпозинионировать окно в необходимом положении. Окно не будет "прилеплено", оно просто будет выровнено в момент вызова stick в указанном положении
		value1 - "left"|"center"|"right";
		value2 - "top"|"center"|"bottom";
	folding(Block_name) - указать блок по клику на который окно будет сворачиваться.
	fold() - свернуть окно
	unfold() -развернуть окно
	first() - текущее окно на переднйи план
	static() - сделать окно статичным, то есть его не льзя будет передвигать мышью, при событии resize оно будет перестаиваться на позицию указанную в stikc. По умолчанию позиция center,center
	mobile(Block_name) - сделать окно мобильным. Его можно будет перемещать зажатием мышки. Если не указан Block_name то перемещать можно будет за любую часть окна, но тогда не будет возможности выделения текста. Если указан Block_name, то перемещение окна будет только за этот блок
	hide() - спрятать окно. Окно не будет разрушено просто будет скрыто
	show() - Отобразить спрятанное окно
	setNewWindowBar(new_windowbar) - Прикрепить текущее окно к дургой панели дял сворачивания
        	new_windowbar - уже созданный обьект панели
	
	UserResize(this_Window) - калбек вызываемый по событию ресайза
