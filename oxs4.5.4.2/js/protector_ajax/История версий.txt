Библиотека protector_ajax
	
	Предназначена для безопaсной работы с ajax, работает в паре с библиотекой protector




php методы:
	
	function GetObject($Name="js_protector_ajax") - создает js обьект для работы с ajax, передает в него все необходимые паарметры для защиты. Создавать обьекты рекомендуеться только через это тметод.

js методы и свойства:
	this.Success = function(Msg) - Функция вызываеться при удачном приеме данных. Msg данные
	this.Error = function() - Функция вызываеться при ошибке в работе с ajax.
	this.POST = function (Path,Data) - фукнция осуществялет POST запрос к файлу Path и передает даные Data
